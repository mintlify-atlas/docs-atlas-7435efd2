# Welcome to TokenStash

TokenStash is a Claude Code plugin that caches project context locally — so Claude starts every session already knowing your codebase. No re-reading files. No re-explaining architecture. Just start coding.

## What Gets Stashed

TokenStash saves summaries of your project as you work, stored in `.tokenstash/`:

<CardGroup cols={2}>
  <Card title="architecture.md" icon="sitemap">
    High-level codebase overview — structure, patterns, key components
  </Card>
  <Card title="decisions.md" icon="clipboard-list">
    Decision log — what was decided each session and why
  </Card>
  <Card title="rules.md" icon="book">
    Project-specific rules (CLAUDE.md on steroids)
  </Card>
  <Card title="files/" icon="folder">
    Per-file summaries (3-5 lines each)
  </Card>
</CardGroup>

## How It Builds Up

TokenStash grows organically as you work:

- **Auto:** After every file edit, Claude updates that file's summary silently
- **Manual:** Run `/stash-save` at the end of a session to save decisions and architecture
- **Full index:** Run `/stash-scan` once on a new project to summarize everything upfront
